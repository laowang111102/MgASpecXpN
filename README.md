# 美食推荐系统 python804

## 项目概述

本项目是基于Python开发的美食数据分析及推荐系统，融合了数据分析、机器学习、Web应用等技术，致力于为用户提供个性化的美食推荐服务。

## 技术栈

- 数据获取：Python爬虫
- 数据处理：pandas
- 数据存储：MySQL
- 数据分析：Python pandas
- 数据可视化：ECharts
- 推荐算法：基于用户的协同过滤算法(UserCF)
- Web应用：Django框架

## 功能模块

### 数据处理

1. 数据获取：从ecook.cn网站爬取美食数据
2. 数据清洗：使用pandas处理重复数据、缺失值，并进行数据格式标准化
3. 数据存储：将清洗后的数据存储到MySQL数据库中

### 数据分析

1. 作者排行
2. 菜系分类统计
3. 评论数量排行
4. 收藏数量排行

### 数据可视化

1. 用户注册趋势图
2. 美食类型分布图
3. 词云图

### 推荐算法

基于用户的协同过滤算法(UserCF)，根据用户收藏行为建模，为用户推荐相似兴趣用户喜欢的美食。

### Web应用

提供以下功能：

1. 用户注册登录
2. 美食浏览
3. 详情查看
4. 评论互动
5. 收藏管理
6. 个性化推荐

## 系统角色

- 用户：注册、登录、浏览美食、评论、收藏、获取个性化推荐等
- 系统管理员：负责系统运维、数据处理、分析及可视化等
